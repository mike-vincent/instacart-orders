instacart-orders

What does instacart-orders do?

instacart-orders exports your Instacart order history to JSON, CSV, or text. instacart-orders outputs order dates, totals, retailers, and full item details.

How do I install instacart-orders?

Install instacart-orders with Homebrew:

brew tap mike-vincent/instacart-order-history-scraper
brew install instacart-orders

How do I get my session cookie?

Get the session cookie from your browser. Log into instacart.com, open DevTools, go to Application → Cookies, and copy the value of _instacart_session_id.

How do I use instacart-orders?

Run instacart-orders with your session cookie:

instacart-orders -s "YOUR_COOKIE"

Examples

# Export all orders as JSON
instacart-orders -s "COOKIE"

# Last 6 months as CSV
instacart-orders -s "COOKIE" --months 6 -f csv

# Last 30 days to a file
instacart-orders -s "COOKIE" --days 30 -o orders.json

# Since January 2024
instacart-orders -s "COOKIE" --since 2024-01-01

# Human-readable text output
instacart-orders -s "COOKIE" -f text

# Quiet mode (no progress output)
instacart-orders -s "COOKIE" -q > orders.json
