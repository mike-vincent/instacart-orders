instacart-scraper

A command-line tool for scraping your Instacart order history.

What does instacart-scraper do?

instacart-scraper exports your Instacart order history to JSON, CSV, or text. instacart-scraper outputs order dates, totals, retailers, and full item details.

How do I install instacart-scraper?

You install instacart-scraper with Homebrew:

brew tap mike-vincent/instacart-scraper
brew install instacart-scraper

How do I get my session cookie?

You get the session cookie from your browser. Log into instacart.com, open DevTools, go to Application → Cookies, and copy the value of _instacart_session_id.

How do I use instacart-scraper?

You use instacart-scraper by running it with your session cookie:

instacart-scraper -s "YOUR_COOKIE"

Examples

# Export all orders as JSON
instacart-scraper -s "COOKIE"

# Last 6 months as CSV
instacart-scraper -s "COOKIE" --months 6 -f csv

# Last 30 days to a file
instacart-scraper -s "COOKIE" --days 30 -o orders.json

# Since January 2024
instacart-scraper -s "COOKIE" --since 2024-01-01

# Human-readable text output
instacart-scraper -s "COOKIE" -f text

# Quiet mode (no progress output)
instacart-scraper -s "COOKIE" -q > orders.json
